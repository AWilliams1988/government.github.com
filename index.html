What is Dockside?

Dockside is a browser-based containerised development environment (CIDE) — a software development tool that let dev teams code within development containers.

{% octicon terminal height:28 class:"fill-blue d-inline mr-2" aria-label:terminal %} Easily Work From Anywhere

Dockside is lightweight and accessible within a browser, making working from anywhere and almost any device possible.

{% octicon server height:28 class:"fill-blue d-inline mr-2" aria-label:server %} Fast and Lean

Launch or clone a devtainer within seconds, and with only a couple of clicks.

{% octicon tools height:28 class:"fill-blue d-inline mr-2" aria-label:tools %} Instant Staging

Stage any devtainer’s services instantly so they can be accessed over HTTPS, privately or on the public internet, whether for review, sign-off or testing, by colleagues, clients or management.

{% octicon bug height:28 class:"fill-blue d-inline mr-2" aria-label:bug %} Completely Eliminate Errors

With Dockside you do not have to worry about differences between development and production environments.

{% octicon lock height:28 class:"fill-blue d-inline mr-2" aria-label:lock %} Safe and Powerful

Provide power users with access to system resources and runtimes, architectures and images, while locking down access for other users through a role-based access control scheme.

{% octicon code height:28 class:"fill-blue d-inline mr-2" aria-label:code %} Low Footprint

Dockside’s resource requirements are mimimal. It is proven to support large numbers of devtainers on relatively small servers, VMs or cloud instances.
